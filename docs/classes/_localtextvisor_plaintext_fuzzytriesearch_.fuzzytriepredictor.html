    • Optional cacheCutoff: number

      If not undefined, then this class caches results for inputs with cacheCutoff or fewer characters.

    • Default value cacheSize: number = 1000

      This limits the size of the cache.

    • Optional abortableCnt: number

      If this is set to a positive integer, then any prior predict computations will be immediately cancelled if a subsequent predict call is made. The prior predict call will return a rejected promise.
